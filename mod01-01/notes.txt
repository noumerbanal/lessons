Git
	Setup
		1: cd <foldername> (Go to directory)
		2: git init (Initializes a connection between your local and repository)
		3: git add -A (Add all files and directories from local storage)
		4: git status (To check which files are added)
		5: git commit -m "<message>" (Commits all changes done/added/removed)
			Note: <message> should be descriptive and present tense
		6: git remote add <alias> <link>
			Ex: git remote add origin https://
		7: git push <alias> master
			Ex: git push origin master

	Push
		1: git add -A
		2: git status  
		2: git commit -m "<message>"
		3: git push <alias> master

	Pull
		1: make the directory
		2: git init
		3: git remote add <alias> <link>
		4: git pull <alias> master

	Future Pull 
		1: Go to same project folder
		2: git pull <alias> master

	SOP
		Working in Class
		1: (Start of Day/Night) Pull from gitlab/github
		2: Work
		3: (Start of Day/Night) Push to gitlab/github
